Community-maintained fan database for **One Piece Treasure Cruise**, continued from the original OPTC Database.

**Live site:** [2shankz.github.io/optc-db.github.io](https://2shankz.github.io/optc-db.github.io/)

## Features

- **Character Database** — Browse, search, and filter every OPTC unit. View stats, captain abilities, specials, sailors, limit breaks, support effects, evolution chains, tags, type/class variants, and more. Sortable/filterable table with a character log for collection tracking.
- **Damage Calculator <span style="color:#e67e22">[UNDER CONSTRUCTION]</span>** — Build a team and calculate damage per turn against each type 
- **Drops Table** — Farm locations for evolvers, skillbooks, and items.
