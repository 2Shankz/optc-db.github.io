Instructions
Damage Window:
  • CTRL: Changes Miss>Below Good>Good>Great>Perfect
  • SHIFT: Changes Opponents Type
  • Drag&Drop: Changes Attack Order

Units Window:
  • CTRL: Changes Orbs Matching>Opposite>None
  • SHIFT: Silences Captain
  • ALT: Locks Units
  • SHIFT+ALT: Disables Unit
  • Drag&Drop: Change Unit Position

Ship Window:
  • CTRL: Opens Ship Selection Window

Defense Window:
  • CTRL: Opens Defense Selection Window
